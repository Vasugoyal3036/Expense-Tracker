### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)

### Installation

1. **Clone or navigate to the project directory:**
   ```bash
   cd project
   ```

2. **Create a virtual environment (recommended):**
   ```bash
   python -m venv venv
   
   # Windows
   venv\Scripts\activate

   source venv/bin/activate
   ```

3. **Install dependencies:**

   ```bash
   python app.py
   ```

   ```

   ```
